# Zomato Clone Project Update List

## Roles and Features Roadmap

### 1. User (Buyer)
- Can buy products (place orders).
- Can browse pages and view listed restaurants/products.

### 2. Vendor (Seller)
- Can list/add products (upload image, set name, category, stock, description, and price).
- Manage vendor specific actions:
  - Add to cart
  - Product management
  - Order management
  - Wallet integration

### 3. Admin (Administrator)
- Verification of vendors.
- Payment verification.
- Refund/return request handling.
- Feedback monitoring.
- User blocking capability.
- Payment mode selection for accounts.

> **Project Structures to Initialize:**
> - `npm create vite@latest user`
> - `npm create vite@latest vender`
> - `npm create vite@latest admin`

### 4. Driver & Delivery Partner
- Track order updates (status: pending, received, out for delivery, delivered).
- Locate nearest delivery partner.
- Order duration estimations.
- Delivery path routing/navigation.
